# Phase 7 — Backend API Layer

Runnable implementation: **`backend/`** at the repository root (FastAPI).

## Quick start

From the repo root (with `.venv` active and `pip install -e ".[api]"`):

```bash
python -m uvicorn backend.app.main:app --reload --host 127.0.0.1 --port 8000
```

See also **`../../backend/README.md`** for environment variables and OpenAPI URL.
